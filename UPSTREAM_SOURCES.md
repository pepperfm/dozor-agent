# Upstream sources used

These public source files from `laravel/nightwatch` were used as the starting point or reference:

- `composer.json`
- `config/nightwatch.php`
- `src/NightwatchServiceProvider.php`
- `src/Ingest.php`
- `src/Payload.php`
- `src/RecordsBuffer.php`
- `src/Console/AgentCommand.php`
- `src/Console/StatusCommand.php`
- `agent/helpers.php`

This bundle intentionally reworks naming, defaults, public commands, and the runtime model
for a self-hosted product called **Dozor**.
