# Dozor Agent brand pack

This pack defines the **supporting mark** for Dozor Agent.

## Chosen concept
**Beacon Tower** - a geometric watchtower / beacon that sends a calm signal.

It should feel like the local watcher that stands near the application and sends health, trace, and anomaly signals to Dozor.

## Recommended use
Use this mark for:
- package pages
- installation docs
- setup wizard screens
- integration cards
- host/source cards
- agent status states
- terminal-friendly onboarding illustrations

## Recommended slogan
**The node that keeps watch locally**

## Files
- `svg/` - vector assets
- `png/` - quick raster exports
- `tokens/` - color tokens and CSS vars
- `usage.md` - usage rules
- `copy.md` - brand voice and copy hooks
