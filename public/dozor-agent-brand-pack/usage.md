# Dozor Agent usage rules

## Role
Beacon Tower is a supporting mark, not the public master brand.

Use it when the UI is talking specifically about the installed agent, source connectivity, local collection, or heartbeat status.

## Good surfaces
- install docs
- source pages
- environment cards
- agent connection status
- package README
- setup modals

## Avoid
Do not replace the main Dozor mark with the tower in the app navbar, landing hero, or favicon unless the surface is clearly about the agent only.
